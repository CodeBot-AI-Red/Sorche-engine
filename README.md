# Sorche Engine (Godot 4.6.2)

Projeto base criado para iniciar a **Sorche Engine** com Godot 4.6.2.

## Estrutura

- `project.godot`: configurações principais do projeto.
- `scenes/Main.tscn`: cena inicial exibindo identidade da engine.
- `scripts/Main.gd`: script de bootstrap da cena principal.
- `icon.svg`: ícone do projeto.

## Como abrir

1. Abra o **Godot 4.6.2**.
2. Clique em **Import**.
3. Selecione o arquivo `project.godot` deste repositório.
4. Execute a cena principal.
